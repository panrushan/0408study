# 🌿 安肌成分检测

一款帮助敏感肌用户检测化妆品成分安全性的Web小程序。

上传化妆品成分表照片，自动识别并分析风险成分，给出个性化安全评分。

## ✨ 功能特点

- 📷 **OCR图片识别**：接入百度OCR，自动识别成分表文字
- ✂️ **智能裁剪框**：上传图片后可框选成分表区域，提高识别精度
- 🌐 **中英成分识别**：英文成分自动匹配中文名
- 📊 **安全评分**：0-100分安全评分，成分按风险等级标色
- 👤 **肤质档案**：记录肤质类型、过敏成分、特殊状况
- ⚡ **成分冲突检测**：自动识别成分间的相互作用风险
- 📖 **成分百科**：点击任意成分查看详情和温和替代建议
- 📜 **历史记录**：保存检测记录，随时回顾

## 🛠️ 技术栈

- 前端：HTML + CSS + JavaScript（原生，无框架）
- 后端：Node.js + 原生 http 模块
- OCR：百度智能云通用文字识别 API

## 🚀 本地运行

### 1. 环境准备

- 安装 [Node.js](https://nodejs.org/)（建议 v18+）
- 注册 [百度智能云](https://cloud.baidu.com/) 账号，开通「通用文字识别」服务
- 创建应用，获取 `API Key` 和 `Secret Key`

### 2. 配置密钥

```bash
# 复制环境变量模板
cp .env.example .env
```

编辑 `.env` 文件，填入你的百度API密钥：

```env
BAIDU_AK=你的API_Key
BAIDU_SK=你的Secret_Key
```

### 3. 安装依赖

```bash
npm install
```

### 4. 启动服务

Windows 用户直接双击：

```
start.bat
```

或在命令行中执行：

```bash
node server.js
```

### 5. 访问

浏览器打开：

```
http://localhost:3000
```
